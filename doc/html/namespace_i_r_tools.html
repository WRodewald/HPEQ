float  targetSampleRate,
unsigned int  windowWidth = 64

Function resamples the impulse response. The implementation uses a sinc convolution approach. The concolution is windowed using a hanning window if the impulse response is longer then windowWidth.

Parameters
irthe impulse response
targetSampleRatethe targeted sample rate
windowWidthreturns the width of the hanning window. If the impulse response is shorter then the window width, no window will be used and the convolution will be calculated over all samples.
Returns
the resampled impulse response
