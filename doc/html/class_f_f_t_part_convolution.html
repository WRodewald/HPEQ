Detailed Description

template<unsigned int MaxSize>
class FFTPartConvolution< MaxSize >

FFTPartConvolution implements a simple partitioned FFT where the full impulse response is split in P partitions. The number of partitions is set by #setPartitioningOrder where the order n sets the number of partitions P = 2^N;

The engine implements the frequency delay line approach discussed in Eric Battenberg, Rimas Avizienis 2011 to prevent unnecessary FFT calls.

Parameters
MaxSizemaximum supported impulse response time, needs to be power of 2
